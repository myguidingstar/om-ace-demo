# Om Ace demo

This is the sample code that accompanies a post I wrote:

<http://www.stuttaford.me/2014/08/10/om-interop-with-3rd-party-libs/>

## How to use

```
git clone https://github.com/robert-stuttaford/om-ace-demo
cd om-ace-demo
lein cljsbuild once
```

And then open `index.html`.
